블럭레벨

새로운 줄(문단)로 시작 화명의 전체를 사용하는 태그 p태그, 그룹태그,ul태그

인라인(Inline) Element

p 또는 br 로 줄을 바꾸지 않는 한 한줄에 나타남. span 태그, 에이태그, 이미지태그 화면의 일부를 차지하는 태그

주의 사항 --span 사용 각자 따로

인라인 요소 안에 인라인 요소를 작성하는건 가능하다. 블록레벨안에 블록레벨을 쓸수 있다. 단, p 태그안에 div를 쓰면 오류발생 블록레벨안에 인라인 태그 쓸 수 있다. 인라인레벨안에 블록레벨 태그를 쓸 수 없다.
